# Telegram бот для создания демотиваторов и классических мемов (также GIF/видео)

Простой Telegram-бот, который создает демотиваторы и классические мемы из фотографий, GIF и видео с добавлением текста.

## Что нового
- Убраны все упоминания доната из интерфейса.
- Добавлена поддержка GIF и видео для создания классических мемов (можно добавить верхний/нижний текст и получить анимированный результат).
  - Если отправить GIF или видео, бот попытается наложить текст по кадрам и вернуть результирующий GIF (или MP4, если GIF-выход невозможен).

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Получите токен бота у [@BotFather](https://t.me/BotFather) в Telegram

3. Установите переменную окружения с токеном:
   - Windows (PowerShell): `$env:TELEGRAM_BOT_TOKEN="ваш_токен"`
   - Windows (CMD): `set TELEGRAM_BOT_TOKEN=ваш_токен`
   - Linux/Mac: `export TELEGRAM_BOT_TOKEN="ваш_токен"`

Примечание: для корректной обработки GIF/видео требуется библиотека moviepy. В некоторых средах moviepy может требовать дополнительные системные пакеты (ffmpeg). Убедитесь, что ffmpeg установлен в системе.

## Запуск

```bash
python bot.py
```

## Использование

1. Отправьте боту команду `/start`
2. Отправьте фото, GIF или видео (можно с подписью)
3. Если медиа без подписи, отправьте текст в формате: `Верхний текст|Нижний текст`
4. Получите готовый мем/демотиватор!

## Пример

- Верхний текст: `Когда забыл сохранить проект`
- Нижний текст: `Но git тебя спас`

Отправьте: `Когда забыл сохранить проект|Но git тебя спас`
