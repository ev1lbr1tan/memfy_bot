# Telegram бот для создания демотиваторов

Простой Telegram-бот, который создает демотиваторы из фотографий и текста.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Получите токен бота у [@BotFather](https://t.me/BotFather) в Telegram

3. Установите переменную окружения с токеном:
   - Windows (PowerShell): `$env:TELEGRAM_BOT_TOKEN="ваш_токен"`
   - Windows (CMD): `set TELEGRAM_BOT_TOKEN=ваш_токен`
   - Linux/Mac: `export TELEGRAM_BOT_TOKEN="ваш_токен"`

## Запуск

```bash
python bot.py
```

## Использование

1. Отправьте боту команду `/start`
2. Отправьте фото (можно с подписью)
3. Если фото без подписи, отправьте текст в формате: `Верхний текст|Нижний текст`
4. Получите готовый демотиватор!

## Пример

- Верхний текст: `Когда забыл сохранить проект`
- Нижний текст: `Но git тебя спас`

Отправьте: `Когда забыл сохранить проект|Но git тебя спас`
